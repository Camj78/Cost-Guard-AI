# Screenshots

Required images for Product Hunt, landing pages, and documentation.

## Required Files

| File | Description | Status |
|------|-------------|--------|
| `observability-dashboard.png` | Observability dashboard — token volume, cost, and risk trend charts | Pending |
| `risk-report-page.png` | Public `/report/[id]` page showing RiskScore, drivers, mitigations | Pending |
| `ci-pr-comment.png` | GitHub PR comment generated by CostGuard Action | Pending |

## Capture Instructions

### observability-dashboard.png
1. Start dev server: `pnpm dev`
2. Log in and navigate to `/dashboard/observability`
3. Use seed data: `pnpm tsx scripts/seed-ai-events.ts`
4. Capture at 1440x900 viewport

### risk-report-page.png
1. Create a share link via the dashboard or API
2. Navigate to `/report/<share_id>`
3. Capture at 1440x900 viewport

### ci-pr-comment.png
1. Push a PR with the CostGuard Action configured
2. Screenshot the CostGuard comment from GitHub
3. Crop to show the comment clearly

## Using pnpm shot

Automated capture via Puppeteer:

```bash
pnpm shot:once
```

Screenshots are saved to `temporary_screenshots/`. Move approved captures to this directory.
