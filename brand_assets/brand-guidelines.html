Voice & Tone
Write Like This
Precise, direct, no fluff.
Use exact numbers. "4,218 tokens" not "thousands of tokens."
Confidence through specificity. State facts.
Treat readers as technical operators.
Never Write Like This
Marketing superlatives ("best," "revolutionary").
Emoji in functional UI or documentation.
Vague risk language ("might fail," "could be an issue").
Hedging. State the risk, name the score.
